Timestamp Microservice

In order to use the service, pass a string to the root url of the server as shown in the example below. If the string passed is a valid natural date or a valid unix timestamp, the service will return an object containing the time in both unix and natural date language. If the string is not a natural date, the service will return an object with both of the values being null.

Example Request Usage:

https://myapp.heroku.com/December%2015,%202015

https://myapp.heroku.com/1450137600

Example Output:

{"unix":1134288000000,"natural":"Sun Dec 11 2005"}
